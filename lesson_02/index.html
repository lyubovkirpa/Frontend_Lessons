CSS intro

Cascading Style Sheets - каскадные таблицы стилей

Свойства, обращенные через селектор по тегу, коснутся всех элементов этой группы

Если мы хотим стилизовать конкретный элемент , используем селектор по классу

CSS можно подключить тремя способами:

  • Первый (приотитетный) - создать файл со стилями CSS
  • Второй - внутренний CSS через тег style
  • Третий - инлайновый CSS,указываем стили внутри конкретного стиля

Используем первый способ, но знаем о всех вариантах
